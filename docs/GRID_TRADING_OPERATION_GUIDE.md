## 💡 常见问题解答

### Q1: 如果买单一直未成交怎么办？

```
A: 分情况处理

情况1: 市场上升，价格未跌到买点
→ 撤销买单，比往上挂，追高买入

情况2: 市场下跌，价格继续下跌
→ 不要追单，耐心等待反弹
→ 如果跌破止损价，立即平仓

推荐: 建立新的买单观察期
- 挂单最多持续1周
- 1周内未成交则调整策略
```

### Q2: 网格太密集容易亏损吗？

```
A: 网格密集是特性，不是缺陷

✓ 密集网格的优势:
- 捕捉更多小波动
- 成本均化效果更好
- 适合高频次交易

⚠️ 密集网格的挑战:
- 手续费占比增加
- 需要更大资金量
- 心理管理难度高

建议: 根据资金量选择
- 10万以下: 4-6个网格
- 10-50万: 6-8个网格
- 50万以上: 8-12个网格
```

### Q3: 两会期间大幅下跌怎么办？

```
A: 有备无患的处理

【提前防范】
1. 留足备用金 (总资金的20%)
2. 设置止损点 (-5%)
3. 不要满仓

【应急方案】
1. 触发止损则立即平仓
2. 保护剩余资金
```

### Q5: 卖出时机很难把握，有什么技巧？

```
A: 使用"区间卖出法"

规则: 按照预设的卖出价格，机械式执行
- 不追求卖在最高点
- 只要价格触及卖点就立即挂单
- 避免心理干扰

例如 510050:
价格 3.129 → 自动卖出 (不要等3.15)
价格 3.169 → 自动卖出 (不要等3.20)

优势:
✓ 锁定收益
✓ 降低贪心导致的失误
✓ 让部分资金回笼继续使用
```

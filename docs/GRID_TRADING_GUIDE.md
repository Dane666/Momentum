# 网格交易标的筛选器

## 功能概述

这是一个用于筛选适合网格化交易的A股股票和ETF的工具，特别针对3月份和两会期间的交易特征进行分析。

## 什么是网格交易？

网格交易是一种机械化的交易策略，适合在震荡市场中使用：
- **核心思想**: 在价格区间内设置多个买卖价格"网格"
- **操作方式**: 价格下跌时分批买入，上涨时分批卖出
- **盈利来源**: 赚取震荡市场的波动收益
- **优势**: 纪律性强，减少情绪干扰，自动化程度高

## 筛选标准

### 1. 技术指标
- **年化波动率**: 15%-40% (波动适中，既有利可图又不会过于剧烈)
- **趋势强度**: R² < 0.5 (横盘震荡，无明显单边趋势)
- **流动性**: 日均成交额 > 5000万 (确保能够顺利成交)
- **价格位置**: 在支撑位和阻力位之间的相对位置

### 2. 季节性特征
- **3月份表现**: 分析历年3月份的收益率、波动率和胜率
- **两会期间**: 特别分析3月1日-3月20日的交易特征
  - 收益稳定性
  - 波动是否适中
  - 最大回撤控制

### 3. 综合评分
满分100分，根据以下维度打分：
- 波动率得分 (30分)
- 趋势强度得分 (25分)
- 流动性得分 (20分)
- 区间震荡得分 (15分)
- 两会期间适合度 (10分)

## 使用方法

### 基本用法

```bash
# 筛选ETF（推荐，ETF更适合网格交易）
python main.py --mode grid --target-type etf

# 筛选股票
python main.py --mode grid --target-type stock

# 筛选全部（ETF + 股票）
python main.py --mode grid --target-type all

# 调整筛选参数
python main.py --mode grid --min-score 70 --top-n 10

# 查看网格交易策略指南
python main.py --mode grid_guide
```

### 参数说明

| 参数 | 说明 | 默认值 |
|-----|------|--------|
| `--target-type` | 标的类型: etf/stock/all | etf |
| `--min-score` | 最低评分 (0-100) | 60 |
| `--top-n` | 显示前N个结果 | 20 |

### 输出说明

筛选结果包含以下关键指标：

```
代码         名称        网格评分  年化波动率%  趋势强度  日均成交额
510300      300ETF      85.00     28.50       0.35      125000
```

- **网格评分**: 综合评分，越高越适合网格交易
- **年化波动率**: 过去120天的年化波动率，15-40%为最佳
- **趋势强度**: R²值，越小说明越横盘（<0.3最佳）
- **当前位置**: 在价格区间中的位置（0=底部，1=顶部）
- **3月平均收益**: 历年3月份的平均收益率
- **两会波动率**: 两会期间的日均波动率
- **两会期间适合**: 是否特别适合两会期间进行网格交易

## 3月份和两会为什么重要？

### 历史规律

1. **政策窗口期**
   - 3月份是全国两会召开时间（通常3月3日-15日）
   - 市场对政策预期较多，往往带来波动

2. **资金面特征**
   - 两会前：市场情绪较为谨慎，波动增加
   - 两会期间：观望情绪浓厚，适合震荡交易
   - 两会后：政策方向明确，可能出现方向性行情

3. **板块轮动**
   - 不同年份的政策重点不同
   - 相关概念板块会有明显波动
   - 宽基ETF相对稳定，适合网格

### 策略建议

**两会前布局（2月底-3月初）**
- 选择筛选出的高分标的
- 建立网格交易计划
- 测试参数设置

**两会期间执行（3月1-20日）**
- 严格执行网格策略
- 利用日内波动赚取差价
- 避免追涨杀跌

**两会后调整（3月下旬）**
- 评估策略效果
- 根据政策方向调整
- 考虑是否继续或退出

## 网格交易实战建议

### 1. 网格参数设置

**推荐配置**:
- **网格数量**: 5-10个
- **价格区间**: 基于支撑位和阻力位
  - 下界: 近期低点 - 5%
  - 上界: 近期高点 + 5%
- **单格资金**: 总资金的10%-20%
- **网格间距**: (上界 - 下界) / 网格数量

**示例（假设300ETF当前价格4.0）**:
```python
支撑位: 3.6
阻力位: 4.4
区间: 3.4 - 4.6 (±5%)
网格数: 8
间距: 0.15

买入档: 3.40, 3.55, 3.70, 3.85
卖出档: 4.00, 4.15, 4.30, 4.45
```

### 2. 风险控制

**止损设置**:
- 区间下方设置止损位（通常下方5-10%）
- 例如：区间下界是3.4，止损位设在3.2

**仓位管理**:
- 不要一次满仓
- 保留30%资金应对极端情况
- 单个标的不超过账户的30%

**交易频率**:
- 避免频繁交易（增加成本）
- 建议单格至少2%的价格变动才交易
- 考虑手续费和印花税的影响

### 3. 适合的账户类型

- ✅ **资金量中等** (10万-100万): 足够分配到多个网格
- ✅ **交易成本较低**: 佣金万2.5以下
- ✅ **时间充裕**: 能够盯盘或使用交易软件
- ❌ **资金量太小** (<5万): 网格资金分散后单笔太少
- ❌ **追求暴利**: 网格收益稳健但不会暴涨

### 4. 常见陷阱

⚠️ **单边趋势市**
- 上涨趋势：会过早卖出，踏空收益
- 下跌趋势：不断买入，越跌越亏
- 应对：严格止损，趋势明显时暂停网格

⚠️ **交易成本**
- 频繁交易导致手续费侵蚀收益
- 股票有印花税（0.1%），ETF无印花税
- 建议：优先选择ETF，降低成本

⚠️ **情绪干扰**
- 看到亏损想加仓
- 看到盈利想追涨
- 应对：严格按网格纪律执行

## 技术实现

### 核心算法

1. **波动率计算**
   ```python
   年化波动率 = 日收益率标准差 × √252 × 100%
   ```

2. **趋势强度**
   ```python
   使用线性回归计算R²值
   R²越接近0 → 横盘
   R²越接近1 → 强趋势
   ```

3. **价格区间**
   ```python
   支撑位 = 20日滚动最低价
   阻力位 = 20日滚动最高价
   当前位置 = (当前价格 - 支撑位) / (阻力位 - 支撑位)
   ```

## 常见问题

**Q: 为什么ETF比股票更适合网格交易？**
A: 
- ETF波动相对平稳
- 无个股黑天鹅风险
- 不会停牌
- 流动性好
- 交易成本低（无印花税）

**Q: 如果价格突破网格区间怎么办？**
A:
- 向上突破：可以考虑部分获利了结，保留底仓
- 向下突破：触发止损，考虑暂时离场
- 重新评估市场环境，调整或停止网格

**Q: 网格交易的预期收益率是多少？**
A:
- 年化收益通常在10%-30%
- 与市场波动率相关
- 不追求暴利，重在稳健

**Q: 需要一直盯盘吗？**
A:
- 建议使用交易软件的条件单功能
- 或每天开盘和尾盘各检查一次
- 不需要全天盯盘

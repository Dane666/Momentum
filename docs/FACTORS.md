## 技术因子

### 1. 动量因子 (Momentum)

| 因子名 | 计算公式 | 参数 | 说明 |
|--------|----------|------|------|
| `MOM_5` | `(close - close_5) / close_5` | 5日 | 5日动量 |
| `MOM_10` | `(close - close_10) / close_10` | 10日 | 10日动量 |
| `MOM_20` | `(close - close_20) / close_20` | 20日 | 20日动量 |

**使用场景**: 捕捉股价短期趋势延续性，正动量表示上涨趋势。

---

### 2. RSI (相对强弱指标)

$$RSI = 100 - \frac{100}{1 + RS}$$

其中 $RS = \frac{AvgGain}{AvgLoss}$

| 因子名 | 周期 | 超买阈值 | 超卖阈值 |
|--------|------|----------|----------|
| `RSI_6` | 6日 | 80 | 20 |
| `RSI_14` | 14日 | 70 | 30 |

**使用场景**: 判断超买超卖状态，结合趋势使用。

---

### 3. MACD (平滑异同移动平均)

```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD柱 = 2 * (DIF - DEA)
```

| 因子名 | 说明 |
|--------|------|
| `MACD_dif` | 快慢均线差 |
| `MACD_dea` | DIF的9日EMA |
| `MACD_hist` | MACD柱状值 |

**信号逻辑**:
- 金叉: DIF上穿DEA → 买入信号
- 死叉: DIF下穿DEA → 卖出信号
- 柱状由负转正 → 趋势转强

---

### 4. 布林带 (Bollinger Bands)

$$Upper = MA_{20} + 2 \times \sigma_{20}$$
$$Lower = MA_{20} - 2 \times \sigma_{20}$$

| 因子名 | 说明 |
|--------|------|
| `BOLL_upper` | 上轨 |
| `BOLL_mid` | 中轨 (20日均线) |
| `BOLL_lower` | 下轨 |
| `BOLL_width` | 带宽 = (上轨-下轨)/中轨 |

**使用场景**: 判断波动区间和突破。

---

### 5. 均线系统 (Moving Averages)

| 因子名 | 周期 | 用途 |
|--------|------|------|
| `MA_5` | 5日 | 短期趋势 |
| `MA_10` | 10日 | 短期支撑/压力 |
| `MA_20` | 20日 | 中期趋势 |
| `MA_60` | 60日 | 中期支撑/压力 |

**多头排列**: MA5 > MA10 > MA20 > MA60

---

### 6. 成交量因子

| 因子名 | 计算公式 | 说明 |
|--------|----------|------|
| `vol_ratio` | `volume / MA(volume, 5)` | 量比 |
| `vol_ma5` | `MA(volume, 5)` | 5日均量 |
| `vol_change` | `volume / volume_prev - 1` | 量变化率 |

**使用场景**: 放量突破确认，缩量回调识别。

---

### 7. ATR (真实波动幅度)

$$TR = max(High-Low, |High-Close_{prev}|, |Low-Close_{prev}|)$$
$$ATR_{14} = SMA(TR, 14)$$

| 因子名 | 说明 |
|--------|------|
| `ATR_14` | 14日ATR绝对值 |
| `ATR_pct` | ATR / Close × 100 (百分比) |

**使用场景**: 动态止损设置，仓位管理。

---

### 8. 乖离率 (BIAS)

$$BIAS = \frac{Close - MA_n}{MA_n} \times 100\%$$

| 因子名 | 周期 | 超买阈值 | 超卖阈值 |
|--------|------|----------|----------|
| `BIAS_5` | 5日 | 5% | -5% |
| `BIAS_10` | 10日 | 8% | -8% |
| `BIAS_20` | 20日 | 12% | -12% |

## Alpha合成因子

### 合成公式

$$Alpha = \sum_{i=1}^{n} w_i \times Factor_i$$

其中各因子权重 $w_i$ 可配置。

### 默认权重配置

| 因子类别 | 权重 | 说明 |
|----------|------|------|
| 动量因子 | 0.25 | 趋势延续性 |
| 技术指标 | 0.20 | RSI/MACD等 |
| 量价因子 | 0.20 | 成交量确认 |
| 资金流向 | 0.20 | 主力资金 |
| 情绪因子 | 0.15 | 市场情绪 |

### 预设配置

1. **动量优先** (`momentum_first`):
   - 动量权重: 0.40
   - 适用: 强趋势市场

2. **质量优先** (`quality_first`):
   - 技术指标权重: 0.35
   - 适用: 震荡市筛选高质量标的

3. **资金流优先** (`flow_first`):
   - 资金流向权重: 0.35
   - 适用: 跟踪主力资金

4. **均衡配置** (`balanced`):
   - 各因子权重相等 (0.20)
   - 适用: 不确定市场环境

## 资金流因子

| 因子名 | 数据源 | 说明 |
|--------|--------|------|
| `north_flow` | 东方财富 | 北向资金净流入 |
| `main_flow` | 东方财富 | 主力资金净流入 |
| `super_flow` | 东方财富 | 超大单净流入 |

**计算逻辑**:
- 连续3日北向流入 → 加分
- 主力资金占比 > 10% → 加分

## 因子使用指南

### 选股流程

```
1. 基础筛选
   └── 剔除ST、停牌、新股

2. 技术筛选
   ├── RSI < 70 (非超买)
   ├── MACD柱 > 0 或金叉
   └── 价格站上MA20

3. 量能确认
   ├── 量比 > 1.0
   └── 近5日有放量阳线

4. Alpha排序
   └── 按合成Alpha分数排序

5. 最终选股
   └── 取Top 5-10
```

### 因子组合建议

| 市场状态 | 推荐因子组合 | 权重侧重 |
|----------|--------------|----------|
| 牛市初期 | 动量 + 资金流 | 动量 0.4 |
| 牛市中期 | 技术 + 动量 | 技术 0.35 |
| 震荡市 | 质量 + 情绪 | 质量 0.35 |
| 熊市反弹 | 超卖 + 资金 | 超卖 0.4 |

### 因子衰减监控

建议定期检查因子IC值：
- IC > 0.03: 因子有效
- 0.01 < IC < 0.03: 因子一般
- IC < 0.01: 因子失效，需调整
